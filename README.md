# ArcDonate

A decentralized donation platform built on Arc Testnet.

## Overview

ArcDonate allows anyone to send USDC donations directly to a smart contract on Arc Testnet. Donors can include an optional message with every donation.

## Features

- Connect wallet
- Send USDC donations
- Optional donor message
- View total amount raised
- Instant on-chain transactions
- Mobile-friendly interface
